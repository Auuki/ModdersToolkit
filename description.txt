ModdersToolkit is a toolkit for modders, hence the name.

Current Tools
-C# REPL, exaluate and run code interactively
-Hitbox visualizer
-Dust tool
-Item defaults tool

Planned Tools
-Projectile spawner
-WorldGen tools
-Player layer viewer.
-Sound explorer/tester
-NPC Loot validator
-NPC Spawn validator
-NPC kill visualizer
